# editorzero

AI-native, self-hostable documentation and collaboration platform. Humans and AI agents as peer co-editors. API + CLI + MCP + Web UI at full parity.

**Status:** Phase 0 — research and planning. No runnable artifact yet.

## Orient

- [`docs/brief.md`](docs/brief.md) — what this is, invariants, assumptions, open questions
- [`docs/adr/`](docs/adr/) — architectural decisions (populated during Phase 1)
- [`AGENTS.md`](AGENTS.md) — conventions and working rules for contributors

## Roadmap
Phase 0 (orient) → Phase 1 (research + ADRs) → Phase 2 (architecture) → Phase 3 (CI + verification harness) → Phase 4 (feature slices) → Phase 5 (hardening + launch).
